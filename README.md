# MoonSunPower Frontend ✨

MoonSunPower의 프론트엔드 레포지토리입니다.

## 기술 스택

- React
- JavaScript
- CSS Modules
- Node.js
- React Router

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 빌드
npm run build
```

## 프로젝트 구조

```
src/
├── components/     # 재사용 가능한 컴포넌트
├── routes/        # 페이지 컴포넌트
├── assets/        # 이미지 및 정적 파일
└── App.js         # 메인 앱 컴포넌트
```
